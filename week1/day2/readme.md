# Salesforce Platform Assignment

## 1. What is Salesforce Platform?

The Salesforce Platform is a cloud-based platform that helps businesses build and manage applications for customer relationships, sales, service, and business automation. It provides tools for data management, security, automation, analytics, and app development without requiring heavy infrastructure.

# 2. Explain

## App
An App in Salesforce is a collection of tabs, objects, and features grouped together for a specific business purpose.

*Example:*  
- Sales App  
- Service App  
- Student Management App  

## Object
An Object is like a database table used to store data in Salesforce.

### Types of Objects:
- *Standard Objects:* Account, Contact, Opportunity
- *Custom Objects:* Student, Course, Attendance

## Tab
A Tab is a user interface element that provides access to objects, records, dashboards, or web pages in Salesforce.

# 3. Difference: Configuration vs Coding
Configuration is done using Salesforce’s built-in tools, so we can create features without programming.  
Coding is used when we need advanced functionality by writing Apex or Lightning Web Components.
Configuration is simple, quick, and easy to maintain.  
Coding gives more control and customization but requires programming knowledge.
